-v jenkins_home:/var/jenkins_home \

"docker pull redis" type commands wont work inside jenkins container until this setup down below:

get into root user by adding -u 0 flag and then install docker via curl to use docker commands inside jenkins container:
1. docker exec -u 0 -it <container id> bash
2. curl https://get.docker.com/ > dockerinstall && chmod 777 dockerinstall && ./dockerinstall
3. chmod 666 /var/run/docker.sock 
4. exit
5. now log back into jenkins container via. docker exec -it <container id> bash
6. try this command if works then all set. docker pull redis
